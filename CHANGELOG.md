## v0.1.0 - 2026-06-27

Initial public release.

### Added

- Windows 10/11 system tray press-to-talk voice input app.
- Global hotkey recording with configurable presets and custom capture.
- OpenAI-style audio transcription and Aliyun Qwen-ASR compatible STT modes.
- Optional OpenAI-compatible LLM transcript refinement.
- Optional Voice Actions intent routing for clear commands such as web search, opening URLs, opening settings/logs, and launching local apps.
- Always-on-top recording overlay with waveform feedback.
- Clipboard-preserving text injection through Win32 APIs.
- Local AppData configuration with DPAPI-encrypted API keys.
- GitHub Actions tag-based release workflow that builds and publishes `justsay.exe`.
